# Nextra

**Nextra**是Next.js之上的一个框架，可以让您构建以内容为中心的网站。它拥有Next.js
的所有强大功能，还可以轻松创建基于Markdown的内容。

## 快速入门

import { Cards } from 'nextra/components'

要开始使用Nextra，您需要首先选择一个主题：

<Cards num={2}>
  <Cards.Card
    image
    arrow
    title="Documentation theme"

  >
    <>![Documentation theme](/assets/docs-theme.png)</>

</Cards>

如果你想在不使用这些内置主题的情况下使用Nextra，你可以遵
循[自定义主题](/docs/custom-theme)文档。
