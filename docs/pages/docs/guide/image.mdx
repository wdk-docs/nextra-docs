# Next.js Image

在MDX中使
用[Next.js Image](https://nextjs.org/docs/basic-features/image-optimization)的标
准方法是直接导入组件：

```mdx
import Image from 'next/image'

<Image src="/demo.png" alt="Hello" width={500} height={500} />
```

## 静态图像

import { Callout } from 'nextra/components'

<Callout>默认情况下，此功能通过Nextra配置中的`staticImage:true`启用。</Callout>

Nextra支持使用Markdown语法自动优化静态图像导入。您不需要指定图像的宽度和高度，只
需使用`![]()`标记语法：

```md filename="index.mdx"
![Hello](/demo.png)
```

这将加载`public`文件夹中的`demo.png`文件, 并自动用Next.js`<Image>`包装.

<Callout>
  如果您不想通过`public`承载图像，也可以使用`![](../public/demo.png)`从相对路径加载图像。
</Callout>

有了Next.js图像，布局不会发生变化，加载图像时默认会显示一个美丽模糊的占位符：
